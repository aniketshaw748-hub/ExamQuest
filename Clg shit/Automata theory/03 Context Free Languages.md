## Chapter at a Glance

- **Context-Free Languages (CFL's)** played a central role in natural languages since the 1950's, and in compilers since the 1960's.

- **Context-Free Grammars (CFG's)** are the basis of BNF-syntax. Today CFL's are increasingly important for XML and their DTD's.

- **The Language of Grammar:** If $G(V, T, P, S)$ is a CFG, then the language of $G$ is
  $$L(G) = \left\{ \omega \in T^* : S \stackrel{*}{\Rightarrow} \omega \right\}$$
  i.e. the set of strings over $T^*$ derivable from the start symbol. If $G$ is a CFG, we call $L(G)$ a context-free language.

- **Parse Trees:** Parse trees are an alternative representation to derivations and recursive inferences. There can be several parse trees for the same string. Ideally there should be only one parse tree for each string, i.e. the language should be unambiguous. Unfortunately, we cannot always remove the ambiguity.

- **Ambiguity in Grammar / Languages:** Let $G = (V, T, P, S)$ be a CFG. We say that $G$ is ambiguous if there is a string in $T^*$ that has more than one parse tree. If every string in $L(G)$ has at most one parse tree, $G$ is said to be unambiguous.

- **Chomsky Normal Form (CNF):** We want to show that every CFL $L$ (without $\varepsilon$) is generated by a CFG where all productions are of the form
  $$A \to BC, \quad \text{or} \quad A \to a.$$
  Here $A$, $B$, $C$ are variables and $a$ is a terminal. This is called CNF, and to get there we have to:
  - Eliminate **useless symbols**, those that do not appear in any derivation $S \stackrel{*}{\Rightarrow} \omega$, for start symbol $S$ and terminal $\omega$.
  - Eliminate **$\varepsilon$-productions**, that is, productions of the form $A \to \varepsilon$.
  - Eliminate **unit productions**, that is, productions of the form $A \to B$, where $A$ and $B$ are variables.

- **Greibach Normal Form (GNF):** A Context Free Grammar is said to be in Greibach Normal Form if each rule in the grammar has one of the following forms:
  1. $A \to a A_1 A_2 \ldots A_n$
  2. $A \to a$ &nbsp;&nbsp; where $A_i, A \in V_N,\ i = 1, 2, \ldots, n$ and $a \in \Sigma$.

  If $\varepsilon$ belongs to the language of the grammar, the rule $A \to \varepsilon$ (where $S$ is the start symbol) is also permitted.

- **Greibach's Theorem:** Every Context Free Language can be generated by a Context Free Grammar which is in Greibach Normal Form (GNF). The above theorem implies that given a grammar $G$ (which generates the language $L(G)$), the grammar can be converted to an equivalent grammar $G'$ which is in GNF such that $L(G') = L(G)$.

**5. What is Greibach Normal Form (GNF) for Context Free grammar? Convert the following grammar into GNF grammar.** *[WBUT 2011, 2012, 2016]*
$$S \to ABb / a$$
$$A \to aaA / B$$
$$B \to bAb$$

**Answer:**

*GNF definition:* as stated in the Chapter at a Glance above (each rule of the form $A \to a A_1 A_2 \ldots A_n$ or $A \to a$).

For Step-0, we need to add a rule $Y \to b$. So we start the main process of conversion of the grammar to GNF with the rules:
$$S \to ABY \mid a$$
$$A \to aaA \mid B$$
$$B \to bBY$$
$$Y \to b$$

There is no immediate left recursion in the grammar. We select the order $S, A, B, Y$. Clearly there is no need for nonterminal substitution. So, the final grammar in GNF is:
$$Y \to b$$
$$B \to bBY$$
$$A \to aaA \mid bBY$$
$$S \to aaBY \mid bBY\,BY \mid a$$

**10. a) What is parse tree?**
**b) Consider the CFG** $S \to aaB$, $A \to bBb / \varepsilon$, $B \to Aa$. Find the Parse tree for the string $aabbababa$. *[WBUT 2015]*

**Answer:**

a) A parse tree is an entity which represents the structure of the derivation of a terminal string from some non-terminal (not necessarily the start symbol). The definition is as in the book. Key features to define are the $\text{root} \in V$ and $\text{yield} \in \Sigma^*$ of each tree:
- For each $\sigma \in \Sigma$, there is a tree with root $\sigma$ and no children; its yield is $\sigma$.
- For each rule $A \to \varepsilon$, there is a tree with root $A$ and one child $\varepsilon$; its yield is $\varepsilon$.
- If $t_1, t_2, \ldots, t_n$ are parse trees with roots $x_1, x_2, \ldots, x_n$ and respective yields $y_1, y_2, \ldots, y_n$, and $A \to x_1 x_2 \ldots x_n$ is a production, then there is a parse tree with root $A$ whose children are $t_1, t_2, \ldots, t_n$. Its root is $A$ and its yield is $y_1 y_2 \ldots y_n$.

b) **Parse tree** `[diagram]`: A parse tree rooted at $S$ with $S \to aaB$ at the root, the $B$ node expanding via $B \to Aa$ and $A \to bBb$ recursively, producing the terminal frontier for the given string.

**12. What do you mean by unit production? Remove unit productions from the grammar** $S \to AB$, $A \to a$, $B \to C$, $C \to D$, $D \to b$. *[WBUT 2017]*

**Answer:**

**1st Part:** A unit production is a production $A \to B$ where both $A$ and $B$ are non-terminals. Unit productions are redundant and hence should be removed.

Repeat the following steps while there is a unit production:
1. Select a unit production $A \to B$, such that there exist a production $B \to \alpha$, where $\alpha$ is a terminal.
2. For every non-unit production $B \to \alpha$ repeat the following step:
   1. Add production $A \to \alpha$ to the grammar.
3. Eliminate $A \to B$ from the grammar.

**2nd Part:** There are 2 unit productions in the grammar — $B \to C$ and $C \to D$.

For production $C \to D$ there is $D \to b$, so we add $C \to b$ and delete $C \to D$ from the grammar. Now we have $B \to C$, so we add a production $B \to b$ and remove $B \to C$ from the grammar.

Now we get the final grammar free of unit production as:
$$S \to AB, \quad A \to a, \quad B \to b, \quad C \to b$$

Now we can see that $C$ is an unreachable symbol and so to get a completely reduced grammar, we remove $C$ from the CFG. Thus the final CFG is:
$$S \to AB, \quad A \to a, \quad B \to b$$

**14. Define Left Linear and Right linear grammar. Construct grammar for the following FA:** *[WBUT 2017]*

**1st Part:**

**2. a) $E \to E + E \mid E * E \mid a$. Prove that the CFG with this production rule is ambiguous. Remove the ambiguity from this grammar.** *[WBUT 2008, 2019]*

The string $a + a * a$ has two distinct derivation trees (one grouping $a + (a * a)$, the other $(a + a) * a$) `[diagram]`, so more than one parse tree exists for a string generated by the grammar. Hence the given grammar is ambiguous.

*Removing ambiguity* (introducing precedence/associativity via separate nonterminals):
$$E \to E + E \mid E * E \mid a \quad \text{(ambiguous original)}$$
*is replaced by the unambiguous grammar* — see the unambiguous form given in Long Answer Q6 below:
$$E \to E + T \mid T, \quad T \to T * F \mid F, \quad F \to E \mid a.$$

**2. b) $S \to AB$; $A \to a$, $B \to C / b$, $C \to D$; $D \to E$, $E \to a$. Remove the unit production.** Given $L = \{ a^n b^n \mid n \ge 0 \}$. Find a CFG to generate $L^2$. *[WBUT 2008, 2019]*

**Answer:**

Here unit productions are: $B \to C$, $C \to D$, $D \to E$.

And original non-unit productions are: $S \to AB$, $A \to a$, $B \to b$, $E \to a$.

Next, right side of all unit production, replace only terminal symbol, so the productions are: $S \to AB$, $A \to a$, $B \to b$, $E \to a$.

Then the complete productions are:
$$S \to AB, \quad A \to a, \quad B \to a \mid b, \quad E \to a$$

Suppose $w = a^n b^n \in L$. Then $awb = a^{n+1} b^{n+1} \in L$. Hence the required CFG (for $L^2$) is:
$$S \to awb \mid \varepsilon$$

**3. a) Convert grammars to Greibach Normal Form (GNF).** *[WBUT 2009]*
$$\text{i)}\ S \to aSa \mid aSb \mid \varepsilon$$
$$\text{ii)}\ S \to aSB \mid aSbS \mid \varepsilon$$

**Answer:** Strictly speaking, none of the given grammars can be converted to GNF since $\varepsilon$ belongs to the languages of both the grammars. However, we continue by ignoring the $S \to \varepsilon$ productions and adding it to the converted grammar.

i) The converted grammar in GNF is:
$$S \to aSA$$
$$S \to aSB$$

$$A \to a$$
$$B \to b$$

**3. b) Find a reduced grammar equivalent to the grammar** $S \to aAa$, $A \to bBB$, $B \to ab$, $C \to aB$. *[WBUT 2009, 2014, 2017]*

**Answer:** Clearly $C$ is a useless symbol. Eliminating $C$, we get:
$$S \to aAa, \quad A \to bBB, \quad B \to ab$$

Now we replace all occurrences of $B$ on rhs of production using $B \to ab$, getting:
$$S \to aAa, \quad A \to b\,ab\,ab$$

Now we similarly substitute $A$, getting:
$$S \to a\,b\,ab\,ab\,a$$
which is the equivalent reduced grammar.

**4. a) Construct CFG for the following:** *[WBUT 2013, 2016]*
**i) Palindrome for binary numbers**
**ii)** $L = \{ a^n b^? c^? d^m \mid m, n > 0 \}$
**iii)** $L = \{ a^n b^m \mid n \ne m \}$

**Answer:**

i) Consider $L_{pal} = \{ w \in \Sigma^* : w = w^R \}$. For example $otto \in L_{pal}$, $madamimadam \in L_{pal}$. In Finnish language e.g. $saippuakauppias \in L_{pal}$ ("soap-merchant").

Let $\Sigma = \{0, 1\}$ and suppose $L_{pal}$ were regular. Let $n$ be given by the pumping lemma. Then $0^n 1 0^n \in L_{pal}$. In reading $0^n$ the FA must make a loop. Omit the loop; contradiction.

Let's define $L_{pal}$ inductively:
- *Basis:* $\varepsilon$, $0$, and $1$ are palindromes.
- *Induction:* If $w$ is a palindrome, so are $0w0$ and $1w1$.
- *Circumscription:* Nothing else is a palindrome.

CFG's is a formal mechanism for definitions such as the one for $L_{pal}$:
$$1.\ P \to \varepsilon \qquad 2.\ P \to 0 \qquad 3.\ P \to 1 \qquad 4.\ P \to 0P0 \qquad 5.\ P \to 1P1$$
$0$ and $1$ are terminals; $P$ is a variable (or nonterminal, or syntactic category); $P$ is in this grammar also the start symbol; 1–5 are productions (or rules).

ii) A CFL $L$ is inherently ambiguous if all grammars for $L$ are ambiguous.
Example: Consider $L = \{ a^n b^n c^m d^m : n \ge 1, m \ge 1 \} \cup \{ a^n b^m c^m d^n : n \ge 1, m \ge 1 \}$.
A grammar for $L$ is:
$$S \to AB \mid C$$
$$A \to aAb \mid ab$$
$$B \to cBd \mid cd$$
$$C \to aCd \mid aDd$$
$$D \to bDc \mid bc$$
Let's look at parsing the string $aabbccdd$. There are two leftmost derivations:
$$S \underset{lm}{\Rightarrow} AB \underset{lm}{\Rightarrow} aAbB \underset{lm}{\Rightarrow} aabbB \underset{lm}{\Rightarrow} aabbcBd \underset{lm}{\Rightarrow} aabbccdd, \quad \text{And}$$
$$S \underset{lm}{\Rightarrow} C \underset{lm}{\Rightarrow} aCd \underset{lm}{\Rightarrow} aaDdd \underset{lm}{\Rightarrow} aabDcdd \underset{lm}{\Rightarrow} aabbccdd$$
It can be shown that every grammar for $L$ behaves like the one above. The language $L$ is inherently ambiguous.

iii) CFG for $L = \{ a^n b^m \mid n \ne m \}$. If $n \ne m$, then two cases are possible.

*Case 1:* $n > m$, $L_1 = \{ a^n b^m : n > m \}$. If $G_A$ be the CFG for $L_1$ language, $G_A = (V_n^A, V_t^A, P^A, S^A)$ with $V_n^A = \{ S_1, A, S^A \}$, $V_t^A = \{ a, b \}$. Productions $P^A$ are:
$$S^A \to A S_1$$
$$S_1 \to a S_1 b / \varepsilon$$
$$A \to aA / a$$

*Case 2:* $n < m$, $L_2 = \{ 0^n 1^m : n < m \}$. If $G_B$ be the CFG for $L_2$, $G_B = (V_n^B, V_t^B, P^B, S^B)$ with $V_n^B = \{ S^B, S_2, B \}$, $V_t^B = \{ a, b \}$. Productions $P^B$ are:
$$S^B \to S_2 B$$
$$S_2 \to a S_2 b / \varepsilon$$
$$B \to bB / b$$

By combining ($G_A$ & $G_B$) we can write CFG for the above:
$$S \to S^A / S^B \quad \text{where } S \text{ is the start symbol.}$$

**4. c) Define non-generating and non-reachable symbols with example.** *[WBUT 2013]*

Example grammar: $S \to AB \mid a$, $A \to aA$, $B \to b$.
- A variable is **non-generating** if it does not derive any string of terminals. In the above example $A$ is a non-generating symbol.
- A variable is **non-reachable** if the start symbol does not derive a string which contains that variable. If $X \to w$, a symbol $y$ is unreachable if string $w$ does not contain $y$. In the above example $B$ is unreachable after deleting non-generating symbol $A$ from production.

**5. Prove that Context Free Languages are not closed under intersection.** *[WBUT 2014, 2017]*

**Answer:** Consider the two languages
$$L_1 = \{ a^n b^n c^m : n \ge 0,\ m \ge 0 \} \quad \text{and} \quad L_2 = \{ a^m b^n c^n : n \ge 0,\ m \ge 0 \}.$$

There are several ways one can show that $L_1$ and $L_2$ are context-free. For instance, a grammar for $L_1$ is:
$$S \to S_1 S_2$$
$$S_2 \to a S_1 b \mid \lambda$$
$$S_2 \to c S_1 \mid \lambda$$

Alternatively, we note that $L_1$ is the concatenation of two context-free languages, so it is context-free. But
$$L_1 \cap L_2 = \{ a^n b^n c^n : n \ge 0 \},$$
which is not context-free. Thus, the family of context-free languages is not closed under intersection.

> *(The grammar for $L_1$ is transcribed as printed; the source's $S_1$/$S_2$ production listing is loose, but $L_1 \cap L_2 = \{a^n b^n c^n\}$ and the non-closure conclusion are the verified result.)*

**6. a) $E \to E + E \mid E * E \mid a$. Prove that the CFG with this production rule is ambiguous. Remove the ambiguity from this grammar.** *[WBUT 2018]*

**1st Part:** $E \to E + E \mid E * E \mid a$. Say, string is $a + a * a$.

The string has two distinct derivation trees `[diagram]` — *Derivation Tree 1* groups as $a + (a * a)$ (outermost $E \to E + E$), *Derivation Tree 2* groups as $(a + a) * a$ (outermost $E \to E * E$). So, more than one derivation tree exists for the string generated from the grammar. Therefore the given grammar is ambiguous. **(Ans.)**

**2nd Part:** Removing ambiguity:
$$E \to E + T \mid T$$
$$T \to T * F \mid F$$
$$F \to E \mid a$$

**7. b) What is useless production? Eliminate $\varepsilon$, unit and useless production from following grammar:** *[WBUT 2023]*
$$A \to bA \mid Bba \mid aa, \quad B \to aba \mid b \mid D, \quad C \to CA \mid AC \mid B, \quad D \to a \mid \varepsilon$$

**Answer:**

**1st Part:** The productions that can never take part in derivation of any string are called useless productions. Similarly, a variable that can never take part in derivation of any string is called a useless variable.

**2nd Part:**
$$A \to bA \mid Bba \mid aa, \quad B \to aba \mid b \mid D, \quad C \to CA \mid AC \mid B, \quad D \to a \mid \varepsilon$$

*After removing unit production* $B \to D$:
$$A \to bA \mid Bba \mid aa$$
$$B \to aba \mid b \mid a \mid \varepsilon$$
$$C \to CA \mid AC \mid B$$
$$D \to a \mid \varepsilon$$

*After removing null productions* $B \to \varepsilon$ and $D \to \varepsilon$:
$$A \to bA \mid Bba \mid aa \mid ba$$
$$B \to aba \mid b \mid a$$
$$C \to CA \mid AC \mid B$$
$$D \to a$$

*After removing useless productions of variable $C$ and $D$:*
$$A \to bA \mid Bba \mid aa \mid ba$$
$$B \to aba \mid b \mid a$$

**8. Transform the CFG into GNF, given** $G = (\{ A_1, A_2, A_3 \}, \{ a, b \}, P, A_1)$ **and production $P$ as** $A_1 \to A_2 A_3$, $A_2 \to A_3 A_1 \mid b$, $A_3 \to A_1 A_2 \mid a$. *[WBUT 2023]*

**Answer:**
$$A_1 \to A_2 A_3$$
$$A_2 \to A_3 A_1 \mid b$$
$$A_3 \to A_1 A_2 \mid a$$

Check production of the form $A_i \to A_j$ where $i \le j$. The production $A_3 \to A_1 A_2 \mid a$ is not in the given form. In the production $A_3 \to A_1 A_2 \mid a$, $i = 3$ and $j = 1$, where $i > j$. Using substitution, we need to bring the production to satisfy the condition $i \le j$.

Substitute $A_1$ in the production $A_3$:
$$A_3 \to A_2 A_3 A_2 \mid a$$
We substitute the production $A_2$ in the production $A_3$:
$$A_3 \to A_3 A_1 A_3 A_2 \mid a \mid b A_3 A_2$$
The above production is in left recursion. Eliminate Left recursion. Here $\alpha$ is $A_1 A_3 A_2$ and $\beta$ is $a \mid b A_3 A_2$. The below grammar shows CFG after eliminating left recursion:
$$A_3 \to a Z_3 \mid b A_3 A_2 Z_3$$
$$Z_3 \to A_1 A_3 A_2 Z_3 \mid \varepsilon$$

Now the production $A_3$ is in GNF. Using substitution brings back all the productions to GNF.

Substitute $A_3$ to the production $A_2$:
$$A_2 \to b \mid a Z_3 A_1 \mid b A_3 A_2 Z_3 A_1 \mid a A_1 \mid b A_3 A_2 A_1$$

> *(Reproduced as printed from FAT-91/92; the source's variable subscripts in this $A_2$ line are partly faint on the scan.)* `[as printed]`

Now, we substitute $A_2$ to the production $A_1$. The production $A_2$ is in GNF:
$$A_1 \to b A_3 \mid a Z_3 A_1 A_3 \mid b A_3 A_2 Z_3 A_1 A_3 \mid a A_1 A_3 \mid b A_3 A_2 A_1 A_3$$

Now we can substitute $A_1$ in production $Z_3$; the production $A_1$ is in GNF. The reader can continue substituting $A_1$ in $Z_3$. Finally, we got all the productions in GNF.
